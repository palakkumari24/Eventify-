🎉 Eventify

A full-stack web application for planning and managing events, built with JavaScript.

📖 Overview

Eventify makes organizing events simple and stress-free. From birthday parties and corporate gatherings to community meetups, Eventify helps you create, manage, and track events through a clean, responsive interface powered by a robust Node.js backend.

✨ Features

📅 Event Creation & Management – Easily create, update, and delete events.

📋 Event Listings – Browse upcoming events and filter by date or type.

🖥️ Responsive Frontend – Optimized UI for both desktop and mobile devices.

🔗 REST API Backend – Structured server to handle event data and business logic.

📡 Frontend–Backend Integration – Seamless communication between the client and server.

🚀 Getting Started
Prerequisites

Make sure you have the following installed:

Node.js (v14 or above)

npm (comes with Node.js)

🔧 Installation

Clone the repository

git clone https://github.com/palakkumari24/Eventify-.git
cd Eventify-

Set up the Backend

cd "Event Planning Backend"
npm install
node server.js

The backend server will start at http://localhost:3000
 (or your configured port).

Open the Frontend

Navigate to Event Planning frontend/2026 event/ and open index.html in your browser, or serve it with a local server:

# Using VS Code Live Server, or:
npx serve "Event Planning frontend/2026 event"
