# Eventify - Premium Event Planning Website

A modern, responsive event planning website built with **React.js** and **Tailwind CSS**.

## Features

- **Hero Section** - Video background with "Plan Your Dream Event" CTA
- **Service Cards** - Animated cards for Weddings, Corporate Events, Birthdays, Concerts
- **Glassmorphism Navbar** - Transparent navbar that changes on scroll
- **Login/Signup Page** - Side-by-side design with Google & Facebook social login
- **Event Listing** - Filter by Category and search functionality
- **Animated Event Cards** - Hover zoom effect with "Book Now" button reveal
- **Step-by-Step Booking** - Date → Venue → Guests (3-step form)
- **Infinite Logo Carousel** - Partner brands showcase
- **Theme** - Rose Pink (#FF69B4) & Soft Grey (#F5F5F5)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
```

## Tech Stack

- React 19 + Vite
- Tailwind CSS v4
- React Router
- Lucide React (icons)
